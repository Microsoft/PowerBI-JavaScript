Embed your own report

Prerequisites

  • A workspace in the Power BI embedded service.
  • PowerBI report imported to your workspace.

Instructions to get embed token

Once you have a report imported to Power BI workspace, you are ready to embed it.

To embed a report, you need to get an embed-Token. You can create this token in multiple ways.
  • Using PowerBI-Cli tool.
  • From .Net Code using Microsoft.PowerBI.Core package.
  • From NodeJS code using powerbi-api package.
